# 🎓 HorizonTechX Student Grade Tracker

A Java console-based application developed as part of the **Horizon TechX Java Programming Internship**. This project helps manage student grades by allowing users to add student records, display details, and analyze performance.

---

## 📌 Features

- ➕ Add Student
- 📋 Display All Students
- 🏆 Find Highest Marks
- 📉 Find Lowest Marks
- 📊 Calculate Average Marks
- 🎯 Automatic Grade Calculation (A+, A, B, C, D, F)
- 🖥️ Menu-Driven Console Application

---
